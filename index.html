What's rlvm?

rlvm is an open source implementation of the RealLive virtual machine for Linux and OSX. RealLive is a game engine used in Japan to write visual novels, a few of which I care about, and many that I do not. Currently, I'm working on getting three games working: Kanon Standard Edition, CLANNAD and Planetarian. The newer Full Voice Edition of CLANNAD is not supported and probably won't be for a while since VisualArts overhauled big internal chunks of RealLive.exe.

rlvm currently doesn't support translation patches. rlvm version 0.4 was recently released and was the first version where CLANNAD and Kanon were completable. The main planned feature for the next version is support for the Kanon patch. Hopefully support will be completed by the time the CLANNAD translation patch is released.

rlvm is open sourced under the GNU General Public License version 3 (or later). I welcome patches, forking the project on github and other code submissions.
